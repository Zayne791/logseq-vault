# Logseq Vault — Shared Notes

This is a Git-synced Logseq vault for collaborative note-taking.

## Setup

1. Download Logseq: https://logseq.com
2. In Logseq, open **Settings > Git Sync**
3. Set repo to: `https://github.com/Zayne791/logseq-vault`
4. Configure your GitHub token for push/pull
5. Notes sync every 5 minutes automatically

## Usage

- Add notes in `pages/` (permanent notes)
- Daily journals in `journals/`
- Everything auto-commits to GitHub
